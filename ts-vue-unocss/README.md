# ts-vue-unocss

支持以下特性：

- vue
- typescript
- unocss
- eslint & prettier
- autoImport & autoComponent
- pinia
- vue-router
- vitest / cypress

## Usage

```sh
# pnpm
pnpm install
```

### Development

```sh
# pnpm
pnpm dev
```

### Build

```sh
#pnpm
pnpm build
```

### Test

```sh
# e2e
pnpm test:e2e
# unit
pnpm test:unit
```
