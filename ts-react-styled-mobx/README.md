# ts-react-styled-mbox

支持以下特性：

- react
- vite
- typescript
- mbox
- styled-components
- eslint / prettier
- autoImport
- react-router
- cypress / vitest

## Usage

```sh
# pnpm
pnpm install
```

### Development

```sh
# pnpm
pnpm dev
```

### Build

```sh
#pnpm
pnpm build
```

### Test

```sh
# e2e
pnpm test:e2e
# unit
pnpm test:unit
```
